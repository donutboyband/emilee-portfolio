# Emilee Portfolio

A portfolio website built with React, Vite, and TanStack Router.

## Tech Stack

- React 19
- Vite
- TanStack Router
- GSAP (animations)
- Tailwind CSS

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deployment

Deploy to Vercel with the following settings:
- Framework Preset: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
